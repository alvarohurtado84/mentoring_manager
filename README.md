# Mentoring manager platform # 

Dependencies:

* Python2.7
* Django-1.5.1
* South-0.8.1
* Django-less


##How to install##

### Ubuntu ###

```$
pip install Django
```

```$
pip install south
```

```$
pip install django-less
```

##How to run##

```$
cd mentoring_manager
``` 

```$
python manage.py runserver
```
